# 技能设计表

## 技能总览

- **总技能数**：11 个（10 个可携带 + 1 个默认自带）
- **可携带数**：每个玩家可选 **3 个**技能
- **回血**：默认自带技能，不占用技能栏，无需选择
- **普攻**：默认自带，不占用技能栏，无需选择
- **技能栏位**：3 个主动技能 + 1 个回血（固定）+ 1 个普攻（固定）

---

## 技能组合建议

| 组合风格 | 推荐技能 | 思路 |
|----------|----------|------|
| 近战爆发 | 巨剑术 + 震雷枪 + 神速 | 神速贴脸 → 巨剑/震雷输出 |
| 远程消耗 | 激光 + 烈焰斩 + 淬毒刃 | 保持距离，毒+远程磨血 |
| 防守反击 | 无敌之盾 + 回血 + 隐身 | 扛伤害，找机会反击 |
| 控制流 | 冰冻 + 震雷枪 + 神速 | 冻住 → 神速贴脸 → 震雷击退 |
| 持久战 | 淬毒刃 + 强化 + 回血 | 上毒弱化、强化平A、回血续航 |
